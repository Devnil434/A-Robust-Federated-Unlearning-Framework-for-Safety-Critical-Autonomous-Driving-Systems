# 🛡️ UrbanShield-HFL

## Secure, Trust-Aware Hierarchical Federated Learning for Urban IoT Ecosystems

UrbanShield-HFL is a research-grade secure federated learning framework designed for smart city and urban IoT environments.

The framework integrates:

- Hierarchical Federated Learning (HFL)
- Dynamic Trust Scoring
- Byzantine-Resilient Aggregation (Krum + Bulyan)
- GAN-based Adversarial Training
- Quarantine & Recovery Mechanism
- Privacy-Preserving Distributed Learning
